* Data Science Wikiとは
Data Science WikiとはデータサイエンスやAI関連の技術をより身近に使えるようにするためのまとめサイトになります．最先端の研究成果や技術，OSSを紹介することでより広く社会で使ってもらい，ビジネスと研究を近づけるきっかけになればと考えています．
