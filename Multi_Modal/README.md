# Multi Modal モデル

## MMBT (実装中)  
* textとimagesを入力    
* マルチモーダルな事前学習なしに，マルチモーダルな事前学習をおこなったモデルと同等の精度を出力できる

## CLIP (実装中)  
* 意味が近い画像とテキスト同士が近いベクトルになるように学習  
* 画像とテキストの埋め込み表現を取得  
* テキストと類似の画像を検索
